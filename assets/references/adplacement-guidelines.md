# 广告位最佳实践

## 广告单元限制

### 桌面端
- 建议：每页最多 3–4 个广告单元
- 首屏：≤3 个广告单元
- 内容与广告比例应偏向内容

### 移动端
- **硬性限制：同时可见的 AdSense 广告单元不超过 2 个**
- 首屏：≤1 个广告单元
- 超过 2 个移动端广告属于政策风险

## 与交互元素的距离

广告不得放置在距离以下元素 5 行 HTML 范围内：
- `<button>` 元素
- `<input>`, `<textarea>`, `<select>` 元素
- 二维码（文本或图片包含 `qr code`, `qrcode`）
- 导航菜单或汉堡按钮
- 下载按钮或播放按钮

## 禁止的广告位

- 图片或媒体上方（覆盖式广告）
- 弹窗、模态框或插页内
- 404 错误页面
- 登录、注册或感谢页面
- 无内容页面（建设中、占位符）
- 诱导点击（"请点击我们的广告来支持我们"）

## 广告密度

- 广告密度在任何屏幕尺寸下不得超过视口的 30%
- 段落之间优先于浮动边距
- 允许粘性广告，但不得遮挡内容

## 检测方法

统计 `<ins class="adsbygoogle">` 出现次数：
- 每页总数
- 在 `@media (max-width: 767px)` 区块或等效移动端 CSS 内
- 距离检查：检查每个广告单元周围 ±5 行 HTML 是否有交互元素
