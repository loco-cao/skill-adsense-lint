# 流量质量分析方法

## 互动信号（公开可见）

### 正面指标
- 评论区有近期活动
- 社交分享按钮（功能正常，非仅装饰）
- 新闻通讯/邮件订阅表单
- 作者署名及简介（表明对内容的投入）
- 近期发布日期（3 个月内）
- 内部链接（每个内容页面 3 个以上内部链接）

### 负面指标
- 流量交换小部件或脚本
- 自动刷新元标签或 JavaScript 重载循环
- 弹窗下层或重定向链
- 过度标题党但缺乏实质性内容
- 相同模板页面暗示 PBN（私有博客网络）

## 流量来源推断

虽然私有分析数据无法访问，但可从以下方面推断质量：
- 干净的 SEO 结构（站点地图、语义化 HTML、内部链接）暗示有机流量导向
- 出站链接中的 UTM 参数可能表明付费推广活动
- 社交媒体嵌入暗示社交流量多样化
- 新闻通讯存在暗示回访用户基础

## 危险信号模式

| 模式 | 含义 |
|---------|--------|
| `auto-refresh` 元标签或 JS | 人为刷高页面浏览量 |
| 流量交换徽章 | 低质量流量 |
| `ptc`, `paid-to-click` 引用 | 违反 AdSense 政策 |
| 不相关网站间相同页脚 | PBN 信号 |
| 所有内容同一天发布 | 可能为自动生成或批量发布 |

## 自我评估项目

- 网站大部分流量是否来自有机搜索？ ±5 分
- 是否有社交媒体关注或社区证据？ ±3 分
- 网站过去是否被 Google 处罚过？ ±5 分
